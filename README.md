## Publicação no GitHub

1. Crie um repositório chamado `portfolio-anisio` no GitHub.
2. No terminal, dentro desta pasta, execute:

```powershell
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/seu-usuario/portfolio-anisio.git
git push -u origin main
```

3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, selecione:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`

Depois disso, o site ficará disponível em:

`https://seu-usuario.github.io/portfolio-anisio/`

## Ajustes pendentes

- Substituir `seu-usuario` pelo utilizador real do GitHub.
- Substituir `seu-perfil` pelo URL real do LinkedIn.
- Atualizar os links dos projetos quando estiverem publicados.
